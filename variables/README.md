# Notas sobre variables

## Nombres válidos para la variables

Ejemplos de nombres válidos:
• camiones
• numero
• buffer
• a1
• j10hola29
• num_alumnos

Ejemplos de nombres no válidos:
• 1abc
• nombre?
• número
• num/alumnos
